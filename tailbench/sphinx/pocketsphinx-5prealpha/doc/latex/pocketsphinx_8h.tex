Add a word to the pronunciation dictionary. 

This function adds a word to the pronunciation dictionary and the current language model (but, obviously, not to the current F\+SG if F\+SG mode is enabled). If the word is already present in one or the other, it does whatever is necessary to ensure that the word can be recognized.
